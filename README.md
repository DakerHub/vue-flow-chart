# v-flow-chart

## Usage

```bash
$ npm i v-flow-chart
```

```html
<VFlowChart
  :width="500"
  :height="500"
  show-grid
  user-router>
</VFlowChart>
```

```javascript
import VFlowChart from 'v-flow-chart'

export default {
  components: {
    VFlowChart
  }
}
```

## Config

参数名 | 描述 | 类型 | 默认值
---|---|---|---
row 1 col 1 | row 1 col 2
row 2 col 1 | row 2 col 2
